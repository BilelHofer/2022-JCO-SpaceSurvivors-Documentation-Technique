Description détaillée

Classe qui représente le menu du jeu.

Cette class permet de lancer les différents modes de jeu, de présenter une page d'information concernant le jeu, d'afficher les statistiques sur les différents modes de jeu et d'affiché une page de paramêtre où l'on peu, changer ces touches et la langue du jeu.

sortGameList permet de définir quelle partie a été jouer dans quelle mode de jeux.

displayRank permet d'afficher les statistiques. clearRank efface l'affichage des statistiques.

getJson récupère le Json où ce trouve les sauvegardes des parties.

getBestPartSurvive retourne les 3 meilleures parties jouées dans le mode survive.

getBestPartBeat retourne les 3 meilleures parties jouées dans le mode battre.

getTotalTime permet de récupèrer le temps de jeu total.

getTotalKill permet de récupèrer le nombre de kill total.
